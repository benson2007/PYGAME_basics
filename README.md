**_2. What is pygame_**:
  * Pygame提供Display,sound,Music,Image,Text,Event幫助製作遊戲
  * Pygame可以做出2-D小遊戲
  * Pygame偵測使用者使用Keyboard,joystick,mouse控制遊戲
  * Pygame提供許多內建的game objects來製作遊戲
